[# Smart Contact Manager

This is a Spring Boot project that allows users to store and manage their contacts online.

## ✨ Features
- User registration and login
- Add, update, delete contacts
- Secure access using Spring Security
- Responsive UI with Thymeleaf

## 🛠️ Tech Stack
- Java
- Spring Boot
- Thymeleaf
- MySQL
- Spring Security

## 🚀 How to Run
1. Clone the repo
2. Configure `application.properties` with your DB credentials
3. Run using `mvn spring-boot:run`
